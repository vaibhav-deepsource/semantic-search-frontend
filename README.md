# Sourcery


## :rocket: Getting started

1.  Install dependencies
    ```bash
    yarn
    ```
2. Create .env & add OpenAI API Key to .env (just like in .env.example)
3.  Start dev server
    ```bash
    yarn dev
    ```
4.  Visit the page  
    <a href="http://localhost:8080" target="_blank">http://localhost:8080</a>
